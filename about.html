Our Journey
Through Space

From a simple university project to a fully interactive arcade experience.

🚀

The Beginning

Space Shooter started as a small academic project built using HTML, CSS, and JavaScript. The goal was simple: create a playable 2D shooting game inside the browser.

🌌

Visual Evolution

We introduced parallax star backgrounds, glowing UI elements, animated spaceship effects, and modern glass-style interfaces to enhance the immersive experience.

💥

Gameplay Upgrade

The game evolved with particle explosions, rotating asteroid enemies, level scaling difficulty, and smooth mouse + keyboard controls.

🏆

Competitive Spirit

A high score system was added using local storage, encouraging players to improve and beat their previous records.

✨

The Future

Upcoming updates may include boss battles, power-ups, global leaderboards, and advanced AI enemies.
